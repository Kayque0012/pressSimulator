# pressSimulator

Versão estática inicial do simulador de prensa pneumática.

## Publicação

Este projeto funciona diretamente no GitHub Pages, sem instalação.

Arquivos principais:

- `index.html`
- `css/styles.css`
- `js/app.js`

O próximo módulo será o parser de arquivos `.msx` e a tela de mapeamento manual de I/O.
